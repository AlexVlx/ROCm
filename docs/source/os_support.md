# OS Support and Virtualization

## OS Support
ROCm supports the operating systems listed below.
| OS                 | Tested Kernel  |
|--------------------|----------------|
| RHEL 9.1            | 5.14           |
| RHEL 8.6 to 8.7    | 4.18           |
| SLES 15 SP4        |                |
| Ubuntu 20.04.5 LTS | 5.15           |
| Ubuntu 22.04.1 LTS | 5.15, OEM 5.17 |
